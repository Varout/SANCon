Colophon

This site was created in TextMate for OS X, in the haml markup language, and converted into html on-the-fly using ruby 2.0.0dev. Snippets of CSS were taken from the CSS sticky footer.

Where possible, all shiny things have been created using html5/css3, and where not possible, the excellent jQuery javascript library was used to keep everything in line.

Derpy Hooves is copyright Hasbro, vectored by Chiko997. Pinkie Pie is also copyright Hasbro, and was vectored by RegolithX. Rarity is best pony.

Title text is Space Age plus Museo Slab and Sans, body text is the ubiquitous Helvetica or Lucida Grande.

Skeleton is used as a base for the media queries. You should really check it out.

Site crafted lovingly in ten seconds flat by Jan-Yves Ruzicka, kept up-to-date by Jason Auger, and hosted by the wonderful boffins at NearlyFreeSpeech.
